# VC

**A simple bare minimum concept video conferencing and chatting application**

### [Demo](https://vc-clone.herokuapp.com): https://vc-clone.herokuapp.com

Made with:

 - Nodejs
 - Express
 - Socket.io
 - Peerjs

To run this;
 
  - Clone the repo
  
	`git clone https://github.com/anorebel/VC.git`
 
  - Enter the directory
  
	  `cd VC`
 
  - Install dependencies
  
	  `npm install`
or
	  `yarn`
	  
 - Run:
 	- In development:
	 	`npm dev`
	 		or
	 	`yarn dev`
	 	
	 - In production:
	 	`npm start`
	 		or
	 	`yarn start`
